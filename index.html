The Castle Cart

❮ ❯

Context: "The Castle Cart" is a Couch LAN racing game set in a fantasy world with classes and magic. We wanted to create a game that combines racing with a class-based mechanic. Each character has its unique abilities, making the races dynamic and strategic.

Lessons Learned: During the development, I learned how to efficiently manage couch co-op synchronization using Unity and C#. I also got more experience designing class abilities and balancing the gameplay.

Project description: A racing game based in a fantasy world with classes and magic. This is a project I made with my friends from Uni. I mainly worked on the gameplay mechanics and designed most of the classes and their spells.

Mushroom Survivors

❮ ❯

Context: "Mushroom Survivors" is a game where players use enemies they've defeated as allies to defeat more enemies. The idea was to create a twist on the "survivors" genre where the player doesn’t directly attack but manipulates enemies.

Lessons Learned: This project helped me improve my AI programming skills as I needed to design complex enemy behaviors and control the "swarm" of allied enemies. I also gained experience with balancing a game's difficulty curve.

Project description: A survivors-style game where enemies become your allies. I worked on the core gameplay mechanics and designed the boss fight. This project is currently confidential in development.

Technologies used:

  • Unity
  • C#
